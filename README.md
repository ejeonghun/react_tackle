## 프로젝트 개요

명확한 답이 없는 문제 및 갈등의 주제에 대하여 다른 사용자들과 "**투표**" 및 "**배팅**"을 할 수 있는 커뮤니티 서비스

## 프로젝트 목표
갈등의 시대를 살고 있는 현재 대한민국에서 다양한 주제와 의견에 대한 게시글과 투표를 통해 다양한 시각과 의견 공유에 목적을 둠
